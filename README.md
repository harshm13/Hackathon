# 🚀 VendorBridge – Procurement & Vendor Management ERP

## 📌 Overview
VendorBridge is a Procurement & Vendor Management ERP system designed to simplify and digitize procurement workflows for organizations.

It provides a centralized platform to manage vendors, RFQs (Request for Quotations), vendor quotations, approvals, purchase orders, and invoice generation efficiently.

---

## 🎯 Problem Statement
Traditional procurement processes are manual, time-consuming, and lack transparency. VendorBridge solves this by providing a structured, digital workflow that improves efficiency, tracking, and decision-making.

---

## 🧠 Key Features

### 🔐 Authentication
- User Login & Signup
- Role-based access control (Admin, Vendor, Manager, Procurement Officer)

### 📊 Dashboard
- Overview of procurement activities
- Pending approvals, active RFQs, recent invoices

### 🏢 Vendor Management
- Add & manage vendors
- Store GST, contact details, and categories

### 📄 RFQ Management
- Create and manage Requests for Quotations
- Assign vendors and deadlines

### 💬 Vendor Quotations
- Vendors submit pricing and delivery timelines
- Editable quotation system

### ⚖️ Quotation Comparison
- Side-by-side comparison
- Lowest price highlighting
- Smart recommendation system

### ✅ Approval Workflow
- Approve / Reject quotations
- Add remarks and track status

### 🧾 Purchase Orders & Invoices
- Auto-generate purchase orders
- Invoice generation with calculations
- Print / Download functionality

### 🔔 Notifications & Activity Logs
- Real-time updates
- RFQ alerts, approval updates, invoice tracking

### 📈 Analytics & Reports
- Vendor performance insights
- Spending analysis
- Procurement trends

---

## 🔄 Workflow

RFQ Creation → Vendor Quotation → Comparison → Approval → Purchase Order → Invoice Generation → Tracking
